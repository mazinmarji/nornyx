# Nornyx Executive Overview

Nornyx is a context-native agentic engineering language for governed AI software systems.

## Thesis

AI software delivery is moving from simple chat-based coding to semi-autonomous engineering loops. The missing layer is a safe, executable control plane for:

- intent;
- context;
- agents;
- skills;
- tools;
- policies;
- guardrails;
- evals;
- harnesses;
- memory;
- approvals;
- traces;
- incidents;
- self-healing;
- improvement loops;
- evidence.

Nornyx provides that control plane.

## v0.1 product definition

Nornyx v0.1 is not a full general-purpose programming language. It is an executable AI-engineering specification that can generate existing artifacts and validate their consistency.

## v1.0 product definition

Nornyx v1.0 is a checker/generator/runtime for governed AI engineering workflows.

## Long-term product definition

Nornyx can evolve into a broader LLM-native language where normal code, agents, policies, context, memory, and evidence are first-class.
