# GOAL-052 Risk Note

This patch is documentation and example guidance only. It does not change parser behavior, checker behavior, schema routing, schema semantics, runtime execution, live connectors, model calls, package publication, deployment, automatic approvals, or GOAL-100.

Residual risk is user confusion between schema inspection and `.nyx` document checking. The guide separates `nornyx schema` from `nornyx check` and keeps the compatibility default explicit.
