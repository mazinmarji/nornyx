ramp. It can inspect repo signals, recommend a first step, generate a minimal
Lite `.nyx` draft, and validate that draft against the current checker in a

Keep future adoption work scoped. Portal wizard implementation, live LLM
authoring, remote Git writes, installers, and production enforcement need a
separate approved goal.
